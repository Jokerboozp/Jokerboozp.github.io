List接口

Comparable接口

  • 所有可以”排序“的类都实现了java.lang.Comparable接口，Comparable接口中只有一个方法：public int compareTo(Object obj);

  • 该方法：

    • 返回0表示this==obj
    • 返回正数表示this>obj
    • 返回负数表示this<obj

  • 实现了Comparable接口的类通过实现compareTo方法从而确定该类对象的排序方式

  • Comparable定义了一个类的两个对象怎么比较
